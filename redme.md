# 🌸 Iris Flower Prediction System

A professional Machine Learning web application that predicts the species of an Iris flower using flower measurements.
This project is built using **Python**, **Scikit-learn**, and **Streamlit** with a modern and interactive user interface.

---

# 📌 Project Overview
The Iris Flower Prediction System is a Machine Learning application that classifies Iris flowers into three species:
- 🌼 Setosa
- 🌺 Versicolor
- 🌷 Virginica

The prediction is based on:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
